Architecture

Five-layer defense, fail-closed by default.

Each layer runs independently and contributes to a unified risk score. High-confidence detections short-circuit; ambiguous cases escalate to ML or LLM.

LAYER 1

Pattern Matching

736 regex + keyword patterns covering OWASP ASI06 attack categories. Always on.

<1 ms
LAYER 2-ML

Transformer Inference

Fine-tuned DistilBERT exported to ONNX. Catches obfuscation Layer 1 misses.

~5 ms
LAYER 2

LLM Semantic Analysis

Optional Claude-powered analysis for the highest-stakes ambiguous cases.

~200 ms
LAYER 3

Trust-Aware Scoring

Per-source trust scores boost or dampen risk for known channels.

<0.1 ms
LAYER 4

Behavioral Baseline

Per-agent statistical baseline detects anomalous drift in real-time.

<1 ms
INTEGRITY

Memory Snapshot & Rollback

SHA-256 baselines per memory entry — detect tampering, restore last safe state.

OWASP AMG

# Or with ML transformer + observability
$ pip install "memgar[ml,observability]"

$ python -c "
from memgar import Analyzer, MemoryEntry
a = Analyzer()
print(a.analyze(MemoryEntry(content='hi')))"

# Or run the API
$ memgar serve --host 0.0.0.0 --port 8000
